# พื้นฐาน React

## Install
```bash
$ npm install
```

## Run

```bash
$ npm start
```

หลังจาก run app ให้เปิด Browser แล้วใส่ URL: [`http://localhost:3000`](http://localhost:3000)

#### Install

```bash
$ npm i react-bootstrap bootstrap
```

