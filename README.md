# Todo App

## 1. Server

### 1.1 Install

```bash 
$ cd ./server
$ npm install
```

### 1,2 Config

กำหนดค่าที่ไฟล์ [./server/config.js](./server/config.js)

### 1.3 Run Server

```bash
$ npm start
```

### 1.4 Run Parse Dashboard

```bash
$ npm run parse-dashboard
```

## 2. Client

### 1.1 Install

```bash 
$ cd ./client
$ npm install
```

### 1.2 Run Server

```bash
$ npm start
```
